# Smart City Traffic Forecasting using Machine Learning

## Project Overview

This project predicts traffic volume at different city junctions using Machine Learning techniques. The objective is to analyze historical traffic data and forecast future traffic flow to support smart city traffic management and reduce congestion.

## Project Objectives

- Predict traffic volume using Machine Learning.
- Perform Data Preprocessing and Feature Engineering.
- Analyze traffic patterns using Exploratory Data Analysis (EDA).
- Compare multiple Machine Learning models.
- Select the best-performing model based on evaluation metrics.

## Technologies Used

- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Machine Learning Models

- Linear Regression
- Decision Tree Regressor
- Random Forest Regressor

## Dataset

The dataset contains historical traffic information, including:

- DateTime
- Junction
- Vehicle Count
- ID

## Project Workflow

1. Load Dataset
2. Data Preprocessing
3. Feature Engineering
4. Exploratory Data Analysis (EDA)
5. Model Training
6. Model Evaluation
7. Traffic Prediction
8. Save Prediction Results

## Model Evaluation

The models were evaluated using:

- Mean Absolute Error (MAE)
- Root Mean Square Error (RMSE)
- R² Score

Among all models, **Random Forest Regressor** achieved the best performance.
